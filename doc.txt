CSR = Client-side Rendering
-renderizando a tela no cliente (processamento tela do usuário)

SSR = Server-side Rendering
-renderizando a tela no servidor (resultado na tela do usuário)

SSG = Static-site Generation
-renderiza no build e carrega em cache uma parte dos dados dinâmicos (gera páginas estáticas de dados dinâmicos e carrega no cache)
